 Языки программирования семейства Си/Си

Язык программирования — это набор формальных правил, по которым пишут программы. Обычный язык нужен для общения людей, а язык программирования — для общения с компьютером. Как и в любом естественном языке, тут есть лексика — слова, функции и операторы, из которых по правилам синтаксиса составляются выражения. Они имеют чёткий, вполне определённый смысл, понятный компьютеру. Языков программирования огромное множество и к ним относятся все языки программирования семейства Си/си

К языкам семейства Си относятся:

  • С
  • С++
  • Objective-C, применяемый корпорацией Apple
  • C#

Освоение любого из языков семейства С даёт разработчику фундаментальные знания и понимание основ программирования. Если вы знаете один из языков С, то изучить любой другой становится намного проще. И не потому, что синтаксис похож, а из-за системности подхода — Си помогает понять логику и архитектуру компьютеров, операционных систем и сетевых технологий.

Актуальность:

Тема языков программирования семейства Си является актуальной в современном мире, поскольку на нём до сих пор пишут немалое количество программ.

Предмет исследования:

Технологические возможности программ разработки, тестирования, создания программ на языках программирования.

Цель проекта:

Выяснить какие языки из семейства Си на данный момент самые актуальные.

История создания языка программирования С

Язык программирования Си - один из самых популярных и используемых языков в мире. Он был создан в начале 1970-х годов программистом Деннисом Ритчи (Dennis Ritchie) в Bell Labs, компании, занимающейся научными исследованиями в сфере телекоммуникаций.

Ритчи разрабатывал язык Си как усовершенствованную версию языка Би, который был создан в Bell Labs еще в 1967 году. Он ставил перед собой задачу создать язык, который бы имел достаточно высокий уровень абстракции, позволяя программистам лучше управлять ресурсами компьютера и упрощал разработку программ.

Для разработки Си, Ритчи использовал основные принципы языков высокого уровня: декларативность, структурность и генерализацию. Он также интегрировал в Си систему типов данных, что позволило программистам очень четко определять типы переменных, что в свою очередь упростило работу с памятью и сократило количество ошибок.

Это позволило Си получить репутацию более надежного и мощного языка программирования, чем языки низкого уровня, такие как ассемблер. Он также был свободно распространяемым, что позволило ему стать основой для многих других языков программирования, таких как С++, Java и Python.

В настоящее время Си продолжает быть одним из самых важных языков программирования, и его использование до сих пор остается широко распространенным во многих областях, от системного программирования до разработки веб-приложений и научных исследований.

Языки семейства С и их приминение

    1) С и его применение

    1.1 Краткая история: Язык программирования C была разработан в 1969-1973 годах сотрудником Bell Labs Деннисом Ритчи как развитие языка B. Первоначально был разработан для реализации операционной системы UNIX, но впоследствии был перенесён на множество других платформ.

    1.2 Применение: На Си программируются микроконтроллеры. Ядро Linux тоже написано на нем. На этом языке пишут игры для мобильных телефонов и многое другое.

    2) С++ и его применение

    2.1 Краткая история: Язык программирования С++ был создан в начале 1980-х годов, его создатель сотрудник фирмы Bell Laboratories — Бьёрн Страуструп. Страуструп добавил к стандартному языку программирования С возможность работы с классами и объектами, тем самым зародил предпосылки нового, основанного на синтаксисе С, языка программирования.

    2.2 Применение: С++ применяют для создания высоконагруженных сервисов, где важна скорость работы, таких как поисковые и рекламные системы, драйверы и операционные системы, игры и приложения, а также в разработке браузеров, потому что люди любят открывать много вкладок, а потом смотреть на зависающий компьютер. Также на нём можно писать программы любой сложности, в том числе модели машинного обучения и программы для автопилотов.

    3) Objective-C и его применение

    3.1 Краткая история: Objective-C был создан Брэдом Коксом в начале 1980-х в его компании Stepstone. Он пытался решить проблему повторного использования кода. Целью Кокса было создание языка, поддерживающего концепцию software IC, подразумевающей возможность собирать программы из готовых компонентов (объектов), подобно тому, как сложные электронные устройства могут быть собраны из набора готовых интегральных микросхем.

    3.2 Применение: Objective C - язык для разработки мобильных и десктопных приложений, преимущественно использующийся в операционных системах компании Apple.

    4) С# и его применение

    4.1 Краткая история: Язык программирования С# был разработан в 1993—2001 годах группой инженеров компании Microsoft под руководством Андерса Хейлсберга и Скотта Вильтаумота как язык разработки приложений для платформы Microsoft.

    4.2 Применение: C# используется в работе с базами данных и сетью, в программировании логики и так далее. Программисты C# востребованы в банках, небольших компаниях и стартапах, мобильной разработке.

    5) Python и его применение

    5.1 Краткая история: Язык программирования Python был создан Гвидо ван Россумом в 1991 году, впервые был выпущен в 1994 году. Язык был разработан для повышения проектирования приложений, а также для автоматизации задач.

    5.2 Применение: Python применяется для разработки программного обеспечения, веб-приложений, мобильных приложений, анализа данных и научных исследований. Также язык используется для создания компонентов операционной системы, игр, алгоритмов машинного обучения и искусственного интеллекта, а также для создания сценариев и автоматизации повседневных задач.

  2. Простейший консольный калькулятор на языке программирования c#:

    

    using System;

class Program
{
    static void Main(string[] args)
    {
        Console.WriteLine("Введите первое число: ");
        double num1 = Convert.ToDouble(Console.ReadLine());
        Console.WriteLine("Введите операцию (+, -, *, /): ");
        string op = Console.ReadLine();
        Console.WriteLine("Введите второе число: ");
        double num2 = Convert.ToDouble(Console.ReadLine());
        if (op == "+")
        {
            Console.WriteLine(num1 + num2);
        }
        else if (op == "-")
        {
            Console.WriteLine(num1 - num2);
        }
        else if (op == "*")
        {
            Console.WriteLine(num1 * num2);
        }
        else if (op == "/")
        {
            Console.WriteLine(num1 / num2);
        }
        else
        {
            Console.WriteLine("Ошибка: неизвестный оператор");
        }
        Console.ReadKey();
    }
}
    1. Сначала используется ключевое слово using, чтобы подключить системную библиотеку System, которая содержит класс Console, который нужен для ввода-вывода в консоль.
    2. Затем определяется класс Program.
    3. В классе Program определяется метод Main, который является входной точкой программы.
    4. В методе Main сначала выводится сообщение с запросом о вводе первого числа при помощи Console.WriteLine.
    5. Затем используется метод Convert.ToDouble для преобразования введенной строки, содержащей первое число, в тип double и сохранения его в переменной num1.
    6. Аналогично запрашивается оператор и сохраняется в переменной op.
    7. Затем также запрашивается и второе число и сохраняется в переменной num2.
    8. При помощи условного оператора if-else программа определяет, какую операцию нужно выполнить.
    9. Если операция - сложение (op == "+"), программа выводит результат сложения чисел.
    10. Если операция - вычитание (op == "-"), программа выводит результат вычитания чисел и т.д. Для умножения используется знак *, для деления знак /.
    11. Если введенный оператор не соответствует ни одному из приведенных (else), программа выводит сообщение об ошибке.
    12. Затем программа ждет, пока пользователь нажмет любую клавишу, и завершает свою работу.
